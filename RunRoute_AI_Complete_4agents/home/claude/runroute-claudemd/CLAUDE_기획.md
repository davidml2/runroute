# RunRoute AI — 기획 에이전트

## 내 역할
PRD 작성, 기능 정의, 유저 스토리, 로드맵 관리, 경쟁사 분석

## 프로젝트 개요
AI 기반 러닝 루트 추천 앱 (iOS / Android / Apple Watch / WearOS)
새로운 장소에서 원하는 거리와 지형을 자연어로 입력하면 AI가 최적 루트 추천

## 현재 완료된 기획 문서
- PRD v2.0 (docs/RunRoute_AI_PRD_v2.0.pdf) — 19페이지, 24주 로드맵 포함
- TAM/SAM/SOM 시장 분석 완료
- GTM 전략 수립 완료
- 수익화 모델: Free(3회/일) / Pro($9.99) / Team($4.99/인)

## 기술 스택 (개발팀 산출물)
- 백엔드: NestJS 10, PostgreSQL, Redis
- AI: Python FastAPI, LangChain, GPT-4o
- 모바일: SwiftUI (iOS), Jetpack Compose (Android)
- 워치: watchOS SwiftUI, WearOS Compose

## 핵심 유저 페르소나
1. **탐험가 유진** (28세) — 출장지에서 러닝, 안전한 루트 필요
2. **챌린저 민준** (35세) — 마라톤 준비, 고강도 루트 원함
3. **입문자 수아** (24세) — 러닝 시작, 쉽고 예쁜 코스 선호

## 24주 로드맵 요약
- Phase 1 (1-8주): MVP — 루트 추천 + 기본 내비게이션
- Phase 2 (9-16주): 소셜 기능 + Apple Watch / WearOS
- Phase 3 (17-24주): AI 고도화 + 글로벌 론칭
